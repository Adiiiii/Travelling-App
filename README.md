# Travelling-App
A travel app, mostly focused on having a clean UI.
Back-end heavy lifting is done via firebase.

### Installation
Install the dependencies and devDependencies and start the server.

```sh
$ Yarn install
$ cd ios && pod istall
$ cd.. yarn run-ios
```
